Optimize the entire website for technical SEO and AI-engine readability.

Make every page crawlable, indexable, fast, and semantically structured for Google, Bing, ChatGPT, Gemini, Perplexity, and other AI answer engines.

Apply these changes:

1. Use clean semantic HTML:

* One clear `<h1>` per page
* Logical `<h2>` and `<h3>` sections
* Real text instead of text embedded inside images
* Descriptive paragraphs, lists, tables, and FAQ sections
* Use `<article>`, `<section>`, `<header>`, `<main>`, `<footer>`, and `<nav>` properly

2. Add complete SEO metadata to every page:

* Unique title tag
* Meta description
* Canonical URL
* Open Graph tags
* Twitter/X card tags
* Robots meta tag set to index, follow

3. Make all images AI-readable:

* Add descriptive alt text to every meaningful image
* Use descriptive filenames
* Compress images
* Add width and height attributes
* Lazy-load non-critical images
* Do not put important text only inside images

4. Add structured data with JSON-LD:

* Organization schema
* Website schema
* BreadcrumbList schema
* Article or Product or Service schema where relevant
* FAQPage schema for FAQ sections
* LocalBusiness schema if this is a local business

5. Improve content for AI answer engines:

* Add a direct answer summary near the top of each page
* Clearly explain who the page is for, what problem it solves, and why it is trustworthy
* Add FAQ sections with concise, factual answers
* Use natural language questions as headings
* Include author, company, date updated, sources, and contact details where relevant
* Avoid vague marketing text; use specific, verifiable claims

6. Improve crawlability:

* Generate `/sitemap.xml`
* Generate `/robots.txt`
* Ensure important pages are internally linked
* Use clean readable URLs
* Avoid hiding important content behind JavaScript-only rendering
* Make navigation accessible through normal HTML links

7. Improve performance:

* Optimize Core Web Vitals
* Minimize JavaScript
* Use server-side rendering or pre-rendering where possible
* Compress CSS/JS
* Use responsive images
* Ensure mobile-first responsive design

8. Improve accessibility:

* Proper heading order
* ARIA labels only where needed
* Keyboard navigation
* Good color contrast
* Descriptive button and link text
* Form labels for all inputs

9. Add trust signals:

* About page
* Contact page
* Privacy policy
* Terms page if relevant
* Author/company bio
* Reviews/testimonials if real
* Clear location, phone, email, and business details where applicable

10. Add AI crawler compatibility:

* Do not block major search or AI crawlers in robots.txt unless intentionally required
* Make the public content accessible without login
* Ensure pages return proper 200 status codes
* Avoid cloaking or showing different content to bots than to users

After applying the changes, audit the site and report:

* Missing title tags
* Missing meta descriptions
* Missing alt text
* Broken links
* Pages blocked from indexing
* Missing schema
* Slow pages
* JavaScript-rendered content that may not be visible to crawlers
