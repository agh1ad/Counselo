Please perform a complete SEO metadata optimization for the entire Counselo website.

Counselo is an online platform for legal consultations. It is not a law firm and should not be described as one. The website serves users in Saudi Arabia and Syria in both English and Arabic.

Brand naming:
- English: Counselo
- Arabic: كاونسلو

Main goals:
1. Improve organic search visibility.
2. Increase click-through rate from Google.
3. Attract users looking for online legal consultations.
4. Ensure every page has unique and relevant metadata.
5. Avoid duplicate titles and descriptions across country, language, service, and article pages.

Do not make unsupported claims such as:
- “best lawyers”
- “guaranteed results”
- “top law firm”
- “win your case”
- “instant legal solution”

Do not describe Counselo as a law firm. Use wording such as:
- online legal consultation platform
- book an online legal consultation
- connect with experienced legal professionals
- receive professional legal guidance

6. LENGTH AND QUALITY RULES
==================================================

Meta title requirements:
- Prefer approximately 50–60 characters when practical.
- Avoid titles that are unnecessarily long.
- Put the main keyword near the beginning.
- Put the brand name at the end.
- Ensure each title is unique.
- Do not stuff multiple keywords.
- Do not repeat the same country or service unnecessarily.

Meta description requirements:
- Prefer approximately 140–160 characters when practical.
- Make every description unique.
- Include the primary legal service or topic.
- Include the country where relevant.
- Use natural, persuasive language.
- Mention confidentiality, convenience, or online access where appropriate.
- Do not make guarantees.
- Do not copy the first paragraph automatically without editing it.

If a description exceeds the recommended length, shorten it naturally without removing the main service, country, or user benefit.

7. TECHNICAL IMPLEMENTATION
==================================================

Apply the metadata dynamically based on the route and page content.

Ensure every public page includes:
- one unique HTML title tag
- one unique meta description
- one self-referencing canonical URL
- Open Graph title
- Open Graph description
- Open Graph URL
- Open Graph type
- Open Graph image where available
- Twitter card
- Twitter title
- Twitter description
- correct language attribute on the HTML element
- correct hreflang annotations for equivalent language and country versions

Render all SEO metadata in the initial server HTML. Do not rely on client-side JavaScript to add metadata after page load.

==================================================
8. CANONICAL AND HREFLANG
==================================================

Each public page must have a self-referencing canonical URL.

For equivalent localized pages, add reciprocal hreflang tags where a genuine equivalent page exists:

- en-SA
- ar-SA
- en-SY
- ar-SY
- x-default

Do not create hreflang links between pages that are not true equivalents.

The x-default URL should point to the main country/language selection page or the most appropriate neutral landing page.

==================================================
9. AVOID DUPLICATION
==================================================

Audit the entire website and identify:
- duplicate titles
- duplicate descriptions
- missing titles
- missing descriptions
- overly long titles
- overly long descriptions
- titles that only contain the page name and brand
- incorrect country names
- English titles on Arabic pages
- Arabic titles on English pages
- pages incorrectly described as a law firm

Fix all detected issues.
